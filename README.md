# Data Warehouse and Analytics Project

Welcome to the **Data Warehouse and Analytics Project** repository! 🚀
This project showcases an end-to-end data warehousing and analytics solution, covering everything from data warehouse construction to delivering meaningful business insights. Built as a portfolio project, it reflects industry best practices across data engineering and analytics.

---

## 🚀 Project Requirements

### Building the Data Warehouse (Data Engineering)

**Objective**

Construct a modern data warehouse using SQL Server to centralize sales data, supporting analytical reporting and data-driven decision-making.

**Specifications**

- **Data Sources**: Load data from two external systems (ERP and CRM) supplied as CSV files.
- **Data Quality**: Identify and resolve data quality issues before proceeding to analysis.
- **Integration**: Merge both sources into a unified, analytics-friendly data model.
- **Scope**: Work with the most current data only; historical tracking is out of scope.
- **Documentation**: Deliver thorough documentation of the data model for both business and technical audiences.

---

### BI: Analytics & Reporting (Data Analytics)

**Objective**

Build SQL-based analytics to extract actionable insights across:

- Customer Behavior
- Product Performance
- Sales Trends

These insights provide stakeholders with the key metrics needed to support informed, strategic decisions.
